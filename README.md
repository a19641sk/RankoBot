### Make a Bot

먼저 필요한 라이브러리의 공식 사이트 주소는 <http://www.tweepy.org/> 입니다.

document에서는 기본적으로 python2버전대의 문법으로 설명이 되어있는데, 
print 함수의 형태에만 유의하면 아마 큰 문제는 없을것 같습니다.

글 본문과, 예시 코드에서의 설명은 Twitter API 중 [REST API](https://dev.twitter.com/rest/public)만 사용 되었습니다. 

만약 실시간 처리가 (ex. 오는 멘션 실시간 확인하기) 필요하다면[Streaming API](https://dev.twitter.com/streaming/overview)를 참고하면 됩니다.

Tweepy에서 이 API들의 이용은 [REST API](http://tweepy.readthedocs.io/en/v3.5.0/api.html#tweepy-api-twitter-api-wrapper), [Streaming API](http://tweepy.readthedocs.io/en/v3.5.0/streaming_how_to.html)인데 후자의 경우 다소 부족한 부분이 있기에 앞에 있는 트위터 공식 document도 참고할 필요가 있을것 같습니다.
